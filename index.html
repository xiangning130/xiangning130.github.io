RSA解题总结

用CTF-RSA-Tools-master和RSACtfTool这两个工具相当强大了，RSACrfTool在python3环境下运行。

一、已知e、n、c，求m

1、用CTF-RSA-tool-master工具(推荐)

把参数写入rsa.txt文件里，运行

$ python solve.py -i rsa.txt

2、用RSACtfTool工具

把三个参数带进命令行。–uncipher 即为密文c

$ python3 RsaCtfTool.py --uncipher 34940946661152652505384725127371968412412443538119822569171955496417633812245 -e 65537 -n 58487247820009414970716333136883008993558343009835955977602531758977268120179

3、easyRSA_exp.py脚本

这个脚本需要把n分解成p、q后，直接将参数写入脚本

$ python easyRSA_exp.py

在线分解n的网站：

http://www.factordb.com/index.php （推荐先用这个，这个网站是通过数据库查询匹配，数据库里记录了在CTF里面出现过的n分解，速度快）

https://www.alpertron.com.ar/ECM.HTM

esayRSA_exp的代码为

#coding:utf-8
#实现RSA解密
from Crypto.Util.number import *

n= 58487247820009414970716333136883008993558343009835955977602531758977268120179 #模数n
e= 65537
c= 34940946661152652505384725127371968412412443538119822569171955496417633812245 #c是密文
p= 237916789679727912591186300880823195027 #通过在线因式分解网站将n分解出p和q
q= 245830686849558293877312212362586985377

d=inverse(e,(p-1)*(q-1)) #求e在 (p-1)*(q-1)的逆元d　
m=pow(c,d,n) # 解密是c的d次方模n，m求出来即为明文
print"m=",m
