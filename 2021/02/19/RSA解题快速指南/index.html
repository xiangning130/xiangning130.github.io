一、基本的解题

1、已知N、e、c，且N较小易分解。

​ 用CTF-RSA-tool-master工具(推荐)，把N、e、c写入rsa.txt中。

​ $ python solve.py -i rsa.txt

2、已知N、e、c，但N比较大不易分解。

​ 用RSACtfTool (参数 –uncipher -e -n)

$ python3 RsaCtfTool.py --uncipher 34940946661152652505384725127371968412412443538119822569171955496417633812245 -e 65537 -n 58487247820009414970716333136883008993558343009835955977602531758977268120179

3、已知公钥pubkey和待解密文件flag.enc

​ 先用CTF-RSA-tool-master工具 ( 参数 –decrypt –k ）

​ $ python solve.py --decrypt flag.enc --k pubkey.pem

​ 若无法解出，尝试用RSACtfTool ( 参数 –publickey –uncipherfile）

​ $ python3 RsaCtfTool.py --publickey pubkey.pem --uncipherfile flag.enc

4、已知公钥pubkey和待解密文件flag.enc，但n较大不好分解

​ CTF-RSA-Tools-master和RSACtfTool都集成很多Attack方法.

​ RSACtfTool (参数 –publickey –uncipherfile)

​ $ python3 RsaCtfTool.py --publickey pubkey.pem --uncipherfile flag.enc

如果两个工具都无法解出，就要考虑用特殊算法来解题了。

二、利用特殊算法的解题

1、共模攻击

​ 如果有两组待解密文件和key文件，且两组的模N都相同，可以用共模攻击。

​ CTF-RSA-Tools-master：将两组的N、e、c写入share_N.txt，运行：

​ $ python solve.py --i share_N.txt

2、公因数攻击

​ 如果有两组n、e、c，且n不相同，n很大很难分解，且题目提示gcd的话，可能两个n有公因数，可以试试公因数攻击。

​ 用gcd_exp.py（脚本在RSA/exp中）。

​ $ python gcd_exp.py

3、已知p、q、dp、dq、c

​ 用RSA-CRT.py（脚本在RSA/exp中）。

​ $ python RSA-CRT.py

4、多质数RSA

​ 如果n分解出多个质数，则可能是多质数RSA，用multi-factor.py (脚本在RSA/exp中)

​ $ python multi-factor.py

5、m、d、p部分泄露

​ 如果m、d或者p部分位数（高位或低位）泄漏，可以用SAGE脚本（脚本在RSA/exp/sage中），需要安装sagemath才能运行。

​ m泄露：$ sage partial_m_exp.sage

​ d泄露：$ sage partial_d_exp.sage

​ p泄露：$ sage partial_p_exp.sage

三、利用工具解决解题过程中的中间值

1、分解大整数因子

（1）在线分解n的网站

​ http://www.factordb.com/index.php （推荐先用这个，这个网站是通过数据库查询匹配，数据库里记录了在CTF里面出现过的N分解，速度快）

​ https://www.alpertron.com.ar/ECM.HTM （速度慢）

（2）yafu (速度快)

​ 我装在windows中，点击yahu-x64.exe，在命令行里输入

​ factor(21)

（3）SageMath （速度慢）

​ $ sage

​ sage: factor(21)

2、查看公钥.pem文件中的N、e等参数

​ （1）CTF-RSA-tool-master ( 参数 -g –dumpkey -k )

​ $ python solve.py -g --dumpkey -k pubkey.pem

​ （2）RSACtfTool ( 参数 –dumpkey –publickey )

​ $ python3 RsaCtfTool.py --dumpkey --publickey pubkey.pem

3、通过公钥生成私钥

​ 用RSACtfTool ( 参数 –publickey –private )

​ $ python3 RsaCtfTool.py --publickey pubkey.pem --private > prikey.pem

4、已知e、N生成公钥

​ 用RSACtfTool ( 参数 –createpub -e -n )

​ $ python3 RsaCtfTool.py --createpub -e 65537 -n 58487247820009414970716333136883008993558343009835955977602531758977268120179
