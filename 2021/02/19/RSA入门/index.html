RSA是非常经典的非对称加密算法，也称为公钥加密算法。RSA算法广泛地应用在数字签名等密码体系中。它有着一定的理论复杂度，应用广泛，同时也经常和AES、Hash等其他的加密算法配合使用，因此RSA是CTF比赛中经常出现的题目。

RSA原理

不同与传统的加密方式，即加密者和解密者用相同的密钥加解密，RSA算法是有两把密钥，公钥和私钥。公钥，顾名思义是公开的密钥。私钥，是加密或解密者私有的，保密的。公钥加密的密文用私钥可以解开，私钥加密的密文可以用公钥解开。在CTF比赛中，一般是公钥和密文容易得到，需要根据现有条件求私钥或者用其他攻击方式求得明文。flag一般就藏在明文中。

RSA加密的可靠性在于：求得两个大素数以及它们的乘积很简单，但是在不知道这两个大素数的情况下，通过乘积求这两个大素数是非常困难的。

​ 所以这里就有三个关键参数：

​ 大素数 1: p, 大素数 2: q ， 以及这两个大素数的乘积：N （ N=pq ）

​ 然后通过一点关于逆元的数论知识：
$$
ed \equiv 1 mod \phi(n)
$$
​ 即 e 和 d 是 在 模 φ(n) 下互为逆元，即e和d的乘积对 φ(n) 取余数恒等于1。

​ 其中 φ (n) = ( p -1) ( q - 1 )

如果已知φ (n), 通过e求出d是很简单的，用invert函数就可以。但是φ (n)本身并不好求，因为它要通过知道p和q的值来计算。p和q的值在N为大整数是非常难求出来的。所以，对于大家，要做RSA题目要牢记以下参数和公式：

首先，牢记RSA加密中出现的以下几个关键参数：

1、两个大的素数 p 和 q，以及它们的积 n，n 是加解密过程中的模数

2、欧拉函数 φ (n) = ( p -1) ( q - 1 )

3、加密指数 e，和解密指数 d = invert ( e, φ(n) )

4、密文 c，明文 m

5、公钥（ e, n ），私钥（d, n）

接下来，牢记下面两个加密和解密公式：

加密： c= m^e mod n

解密： m=c^d mod n
